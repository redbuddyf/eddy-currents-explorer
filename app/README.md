# ⚡ Eddy Current Explorer

An interactive educational web application exploring eddy currents, electromagnetic braking, and maglev technology.

![Eddy Current Explorer](screenshot.png)

## 🌟 Features

- **Interactive Simulations** - Real-time physics demos
- **Roller Coaster** - Experience magnetic braking on a realistic coaster
- **Maglev Train** - Levitate and propel a train to 603 km/h
- **Virtual Lab** - 5 hands-on experiments
- **Quiz** - Test your knowledge

## 🚀 Quick Start

Open `index.html` in any modern browser, or visit the deployed version.

## 📱 Install as App

This is a Progressive Web App (PWA)! You can:
- **iOS:** Safari → Share → "Add to Home Screen"
- **Android:** Chrome → Menu → "Add to Home Screen"
- **Desktop:** Chrome → Install icon in address bar

Works offline once installed!

## 🛠️ Tech Stack

- HTML5
- CSS3 (Glassmorphism design)
- Vanilla JavaScript (no frameworks)
- SVG for graphics
- Canvas for simulations
